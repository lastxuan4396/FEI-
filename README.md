# FEI- Interactive Web Lab

FEI- is a collection of interactive web experiments built around playful interfaces, browser games, and UI studies.

## Main Subprojects

- `apple-inspired-lab/`: Apple-style visual experiments and interaction studies
- `flight-chess-share/`: Flight chess themed shareable web experience
- `f1-game/`: Racing-inspired browser game prototype
- `interactive-test-site/`: Sandbox for interactive UI exploration

## Local Preview

Most subprojects are static pages. From the repo root, run:

```powershell
cd E:\Windows\Codex\portfolio_repos\FEI-
python -m http.server 4173
```

Then open the subproject you want to inspect in the browser.

## Deploy

The current `render.yaml` deploys `apple-inspired-lab/` as the default static site.
