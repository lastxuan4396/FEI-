这个项目是“垫饥的饭”的第一版内容站，也是后续所有更新的底座。

## 这次解决了什么

以前的版本更像一张静态首页，现在这一版开始具备长期运营的基础能力：

- 用 Markdown 管理文章、网页项目和游戏日志
- 用 Astro 自动生成详情页
- 用 API 路由接真实表单，不再依赖 mailto
- 预留统计配置，后面可以接 Plausible 或 GA4

## 为什么选 Astro

Astro 很适合这种“内容为主、交互适量”的站：

- 页面快
- SEO 友好
- Markdown 管理舒服
- 后面还能慢慢长成交互组件

对独立创作者来说，它不像全栈框架那么重，但扩展性已经够用了。

## 页面结构怎么考虑的

V1 没有追求做花，而是先让信息架构成立：

1. 首页先告诉人你是谁
2. 接着给出想法、网页、游戏三类入口
3. 末尾用订阅和合作表单承接动作

这样既适合内容型用户，也适合潜在合作方。

## <a id="roadmap"></a>后面的升级方向

下一阶段我想继续补这几件事：

- 绑定正式域名
- 给每篇内容增加目录和相关文章
- 做更真实的作品截图和试玩入口
- 接入统计面板，看哪些内容最容易被点开

V1 的目标不是完美，而是先搭起能持续长内容的骨架。
